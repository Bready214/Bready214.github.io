📜 Politique de Confidentialité

Dernière mise à jour : juin 2026

👋 Introduction – Qui sommes-nous ?

Bienvenue sur le site de Peepers Party ! Je suis un développeur solo qui crée ce jeu à des fins de test personnel avec mes amis. Ce projet n'est pas une structure commerciale établie — c'est un projet amateur réalisé pour le plaisir et l'apprentissage.

Cette politique de confidentialité explique comment vos données sont collectées et utilisées lorsque vous utilisez ce site web et le jeu Peepers Party.

🎮 Données collectées via Epic Online Services (EOS)

Peepers Party utilise Epic Online Services (EOS) pour l'authentification des joueurs. Lorsque vous vous connectez via votre compte Epic Games, les données suivantes sont collectées par Epic Games :

  • Identifiant de compte Epic – Un identifiant unique associé à votre compte
  • Nom d'affichage – Le pseudonyme que vous utilisez sur Epic Games
  • Informations de présence – Statut de connexion pour les fonctionnalités multijoueur
  • Données de session – Informations nécessaires pour la connexion et la gestion des sessions de jeu

Important : En tant que développeur solo, je n'ai pas accès direct à vos données de compte Epic. EOS gère l'authentification de manière sécurisée et je reçois uniquement les informations nécessaires pour vous permettre de jouer.

📄 Pour en savoir plus sur le traitement de vos données par Epic Games, consultez leur Politique de Confidentialité officielle.

📥 Données collectées via itch.io

Le jeu Peepers Party est disponible en téléchargement sur itch.io. Lorsque vous téléchargez le jeu depuis cette plateforme :

  • itch.io collecte ses propres données conformément à leur propre politique de confidentialité
  • Les données de téléchargement (historique, méthode de paiement si applicable) sont gérées par itch.io
  • Ce site web intègre un lien externe vers itch.io — nous ne collectons aucune donnée lors de votre navigation vers cette plateforme

📄 Veuillez consulter la Politique de Confidentialité d'itch.io pour plus d'informations sur leurs pratiques.

🍪 Cookies

Ce site web n'utilise aucun cookie tiers pour le suivi ou la publicité.

Le site peut utiliser des cookies techniques nécessaires au bon fonctionnement (préférences linguistiques, paramètres de session), mais ceux-ci ne collectent aucune donnée personnelle et sont immédiatement supprimés à la fermeture du navigateur.

🌐 Données personnelles sur ce site

Aucune donnée personnelle n'est collectée via ce site web.

Vous pouvez naviguer sur ce site, découvrir le jeu et accéder aux liens de téléchargement sans nous fournir aucune information personnelle. Nous ne collectons pas :

  • Votre adresse email (sauf si vous nous contactez volontairement)
  • Votre adresse IP ou données de navigation
  • Vos données de localisation

⚖️ Vos droits – RGPD

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès – Obtenir une confirmation du traitement de vos données et en recevoir une copie
  • Droit de rectification – Demander la correction de données inexactes
  • Droit à l'effacement – Demander la suppression de vos données
  • Droit d'opposition – Vous opposer au traitement de vos données
  • Droit à la portabilité – Recevoir vos données dans un format structuré

Pour les données collectées via Epic Online Services : Veuillez contacter directement Epic Games pour exercer vos droits concernant les données de votre compte Epic.

Pour les données itch.io : Veuillez contacter directement itch.io pour toute demande relative à leurs services.

🔒 Sécurité des données

La sécurité de vos données est importante pour moi. Voici les mesures mises en place :

  • Connexion sécurisée – Le site utilise HTTPS pour toutes les communications
  • Stockage minimal – Je ne stocke aucune donnée personnelle sur mes serveurs
  • Accès limité – Seul le développeur (moi) a accès aux données strictement nécessaires au fonctionnement du jeu
  • EOS sécurisé – L'authentification via Epic Online Services utilise les protocoles de sécurité officiels d'Epic Games

📝 Modifications de cette politique

Cette politique de confidentialité peut être modifiée à tout moment pour refléter des changements dans nos pratiques ou pour des raisons légales.

Quand ? En cas de modification majeure, la date en haut de cette page sera mise à jour.

Que faire ? Nous vous recommandons de consulter régulièrement cette page pour rester informé des changements.
